## Module Creation



A Module enables you to combine a strategy(s) and/or contents with a template. Here you can also configure the number of results to be shown and link the selected strategy(s)/content(s) to specific tiles within the module.

## <a name="_ufu6i3l2xjd"></a>**Getting to Modules Creation Screen**

1. From the Top Navigation Bar, navigate to ‘Digital Experience Hub or open the nested menu
1. Click on ‘Assets’ to view the nested menu
1. Click on ‘Modules’ to land on the Modules Listing Screen
1. Click on the ‘Create Module’ Button to create a new module or ‘Edit’ icon to edit and existing one

## <a name="_e7pvwhx8vyjj"></a>**Create a Module:**

To create a Module, click on the ‘**Create Module**’ Button on the Module Listing Screen. You will be brought to the Module Configuration Screen where you can select the integration mode, template and link recommendation(s). You can also define where each strategy is placed within your module here.

**Module Details:**

Click the ‘**Create Module**’ CTA on the Module Listing Screen
Give your Module a unique name

**Note**: You can save your Module config anytime by clicking on “Continue Later” CTA & clicking on “Save & Exit”. The Module is displayed on the listing screen in a draft state.

### <a name="_baxtk1dqpijl"></a>**Integration Modes**

You can select how you would like to integrate this module using the following options:

1. **Embed Code**: enables you to select a template, strategy(s) and define it’s configurations to render directly on your platform via Embed Code
1. **API**: enables you to select strategy(s) and define it’s configurations to return results via API
1. **Email**: enables you to configure an Email module by selecting a Email template, strategy(s) and define it’s configurations to return results via an HTML snippet

**Note**: Changing the integration mode will reset the module configuration. This field cannot be changed while the Module is Live

### <a name="_gal07g9h6yh2"></a>**Select a Template**

When the integration mode selected is Embed Code or Email, you can select a Template here to display your content within. Click the **‘Select’** button on the Select Template Field to view the Listing of created Templates.

1. You will be able to view and scroll through the list of **Created Templates** to select from
1. The Desktop, Mobile and Email (if applicable) **Preview** Each Template will be displayed on the right panel
1. You can use the **Search or Filter** by Template Type options to help narrow down your selection
1. Once you have selected your Template, click the **‘Confirm’** CTA to return to the Module Config screen where the selected template name will be displayed and it’s preview will appear on the the right panel
1. You can edit your selection by clicking the **‘Change’** CTA

**Note**: Changing the Template will reset the rest of module configuration. This field cannot be changed while the Module is Live.

### <a name="_guv8w2fa9oat"></a>**Module Configurations**

You can define the properties of the module in this section. Module Configuration Options will vary based on the Integration Mode Selected.

#### <a name="_v9mjf3c0silg"></a>**Embed Code Module Configurations**

1. **Module Title**: Provide a Module Title in this section. Any text you input can we previewed on the Right panel Preview Section in the styling format of your selected Template
1. **Min Tiles**: Provide the minimum number of recommendation results you want returned in to render the module on your platform
1. **Max Tiles**: Provide the maximum number of recommended results you want returned and displayed within this module. This field determines the number of Tiles in the module

#### <a name="_1ns4d79p0c1c"></a>**API Module Configurations**

1. **Min Tiles**: Provide the minimum number of recommendation results you want returned in to render the module on your platform
1. **Max Tiles**: Provide the maximum number of recommended results you want returned and displayed within this module. This field determines the number of Tiles in the module

#### <a name="_imgkl74j5t96"></a>**Email Module Configurations**

1. **Module Title**: Provide a Module Title in this section. Any text you input can we previewed on the Right panel Preview Section in the styling format of your selected Email Template
1. **Email Platform**: Select the Email Platform you wish to integrate this module into. This will help unify the HTML snippet format with the provided platform. If you would like to ‘**Add a New Platform’**, click the CTA and enter a Platform Name, User ID Format, Product ID Format and Timestamp Format.
1. **Card Height and Width:** Provide the desired Height and Width (in px) of each content Card. By adjusting the Height, the Width will automatically adjust to the a desired ratio
1. **Number of Rows and Columns:** Provide the number of rows and columns you want to display within each email. These fields determines the number of Tiles in the module

### <a name="_2fvajfnscmyl"></a>**Link Recommendations**

Within the ‘Link Recommendations’ accordion, you will be able to select strategy(s) and/or static content to add to your module. You will also be able to assign the selected to specific tiles to meet your business needs.

#### <a name="_ezik1b4cvva9"></a>**Select Recommendations**

1. Click on the ‘**Add Recommendations**’ CTA to open the Select Recommendations Modal
1. Select Strategy(s) from the first Tab, **‘Strategy’**. This Tab lists all your created Strategies. Use the Search, Filter or Sort options to further narrow your selection. You can multi-select upto 3 strategies to add to your module.
1. Select Content from the second Tab, “**Content’**. Here you will find a listing of all your Content. You can navigate to different Catalogs or Feeds using the provided dropdowns. You can also you the Search, Filter or Zoom options to further narrow your selection. Multi-Select as many as you Content(s) you wish to add to your Module
1. Once you have selected the desired strategies/content, click the **‘Confirm’** CTA and you will see the selected strategies and/or content display within the Link Recommendations Accordion.
1. To view the info, replace or delete any of the linked recommendations, use the respective icons within the ‘Actions’ column.
1. To add or edit your selection, click on the ‘**Manage Recommendations**’ CTA within the Link Recommendations Accordion to return the Select Recommendations Modal

### <a name="_orq4a6c38ly1"></a>**Assign Tiles**

1. Once you have linked recommendations to your module, you can assign them to specific tiles
1. To assign a linked Content, click on the dropdown associated with it under the column “Tiles Assigned” and select the Tile where you would like that content to be placed. You can move it to any other available tile as needed. Once you are happy with the selection, click the ‘Apply’ CTA to save and close the dropdown
1. To assign a linked Strategy, click on the dropdown associated with it under the column “Tiles Assigned” and select a tile or a range of tiles. To select a range, double click on the tile you want to start from and click on the end of your range to select all available tiles in between. You can also click the ‘Select Unassigned’ CTA to select all available tiles. Once you are happy with the selection, click the ‘Apply’ CTA to save and close the dropdown
1. To clear any selected tiles, you can user the “Clear All’ CTA on the column header to clear all selected tiles or you can clear the selection on any linked strategy or content within it’s Tiles Assigned dropdown
1. To see the number of tiles assigned in relation to the total, view the ‘Assigned Tiles’ progress bar. You can also view the count of each linked strategy/content on it’s assigned tiles dropdown

### <a name="_qjk4n4hx0wvy"></a>**Linking Recommendations - 'Actions' Column:**

1. **Assign Fallback Strategy:** In case a strategy returns fewer results than configured items, the fallback strategy will backfill recommended items. You can select one fallback strategy.
1. **Strategy/Content Details:** If strategy linked, this option shows the information of the strategy linked, modules, experiences associated to the strategy. If content linked, it shows the content details - Metadata and Tags associated to the content.
1. **Replace:** Allows to replace the content/strategy that has been linked.
1. **Remove:** Allows to remove the content/strategy that has been linked.

### <a name="_40j4ys52eo9i"></a>**Preview**

You will be able to view a preview of your module on the right panel of the screen. The preview will display your selected template (if applicable), the tile legend of each tile and a preview of your content within the allocated tiles.

1. A template preview will appear as soon as you have selected a template when your selected integration mode is Embed Code or Email. If your selected integration mode is API, you will be able to preview the tile legend as soon as you have select integration mode
1. You will be able to see a tile number associated with each tile on the preview. This will help you while assigning strategies and content to desired tiles
1. Once you have assigned any content to a tile, you will be able to see it on the exact tile selected within the module preview
1. Once you have assigned any strategies to a tile(s), you will be able to see placeholder content on each selected tile within the module preview
1. To view the preview in a full screen view, click the ‘Expand’ icon. To return to the two pane view, click on the ‘Minimize’ icon

### <a name="_mi3mhwzhnjdn"></a>**Create Module**

Once you have set up the desired recommendation module, click on the ‘Create’ CTA at the top of the right corner of the screen.

Once you create a Module, it will be labeled as ‘Inactive’ on the Module Listing Page. Once you link the module to any Published Experience, the status will be reflected as ‘Live’

## <a name="_46vbei6go3st"></a>**Glossary:**

| **Term**           | **Definition**                                                                                                                             |
| :----------------- | :----------------------------------------------------------------------------------------------------------------------------------------- |
| Module             | A Module is a combination of one or more recommendation Strategies with an optional Template                                               |
| Experience         | An Experience is any touchpoint on your website where you have added one or more recommendations and/or set up an Experiment               |
| Tile               | A tile is a placement on your template or recommendation results                                                                           |
| Integration Mode   | Integration mode is the way you would like to integrate the module onto you site; via Embed Code, API or Email                             |
| Continue Later CTA | The Continue Later CTA allows you to save or discard changes made to the Experiment Settings and to return to the Experience Listing Page. |
